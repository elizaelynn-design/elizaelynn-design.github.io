Sample Questions

Practice set

Q1. Which Kubernetes object is responsible for maintaining the desired number of pod replicas?
A. Service   B. Deployment   C. ConfigMap   D. Node
Answer: B — Deployments manage pod replicas and ensure the desired state.
Q2. What is the role of kubectl in Kubernetes?
A. It monitors logs   B. It provisions cloud infrastructure   C. It interacts with the Kubernetes API server   D. It stores container images
Answer: C — kubectl is the CLI tool for communicating with the API server.
Q3. Which CNCF project focuses on observability and metrics collection?
A. Prometheus   B. Envoy   C. Helm   D. Fluentd
Answer: A — Prometheus is widely used for metrics and monitoring.
Q4. What does a Kubernetes Service do?
A. Stores secrets   B. Provides persistent storage   C. Exposes pods to network traffic   D. Manages container images
Answer: C — Services expose pods and enable stable networking.
Q5. Which component helps automate application delivery in cloud native environments?
A. CI/CD pipeline   B. Pod autoscaler   C. Node controller   D. Scheduler
Answer: A — CI/CD pipelines streamline deployment and delivery.
